# 5 Minute Habits 🌱

A simple motivational web application designed to help people replace mindless scrolling with small intentional actions.

## 🚀 Features

* 5 minute focus timer
* Habit categories (Mind / Body / Growth)
* Custom habit creation
* Streak tracking system
* Progress calendar for the last 30 days
* Anti-scroll reminder
* Distraction counter
* Dark mode

## 🧠 Concept

The idea is inspired by the principle that **small actions create long-term change**.
Instead of spending time scrolling on social media, users are encouraged to start with just **5 minutes of focused effort**.
